Introduction
============

This :term:`VirtualBox` VM (Virtual Machine) Appliance is one of the fastest and easiest ways to review or evaluate :term:`Plone`.

It is a self-contained appliance that will run independent from your current operating system.

This virtual machine contains a minimal Linux operating system and a pre-configured :term:`Plone` stack.

.. note:: **This appliance is neither meant nor suited production environments !**
